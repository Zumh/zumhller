# zumhller
This is for my resume, blogs and portfolio
- work in progress for projects

[zumh.github.io](https://zumh.github.io/zumhller/)

## Useage 

### create posts
- hugo new posts/my-new-post.md
- hugo server -D
- open http://localhost:1313/posts/my-new-post/

### create content
- hugo new content my-new-post.md
- hugo server -D
- open http://localhost:1313/content/my-new-post.md

## remove draft
- draft: false
